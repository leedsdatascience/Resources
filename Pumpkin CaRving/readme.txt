We hope you enjoy this Halloween workshop. 
Instructions are in the Pumpkin-CaRving.pdf file. 
The file Pumpkin-CaRving.R contains all the R commands from the workshop.
The data is in the file pumpkin.csv. 
You can download the data by clicking on the csv file in github then right clicking on the 'Raw' button and choosing 'save link as...'
Choose 'all files' under 'Save as type' in the save window and call your file 'pumpkin.csv'.
Make a note of where you've saved it!
